# SQL Mini Projects

A collection of structured SQL mini projects built using PostgreSQL to practice database design, data insertion, joins, aggregate functions, subqueries, and business-related SQL analysis.

This repository demonstrates practical SQL concepts through real-world database systems such as Employee Management, Library Management, and Sales Management systems.

---

## Technologies Used

- PostgreSQL
- SQL

---

# Projects Included

## 1. Employee Management System

A relational database project designed to manage employee records, departments, salaries, and job roles.

### Features

- Table creation with Primary & Foreign Keys
- Employee and department management
- JOIN queries
- Aggregate functions
- GROUP BY operations
- Subqueries
- Views for reporting
- Salary analysis

### SQL Concepts Used

```sql
JOIN
GROUP BY
ORDER BY
COUNT()
SUM()
AVG()
SUBQUERY
VIEW
```

## 2. Library Management System

### Features
