# Sales Management System

A structured SQL database project built using PostgreSQL to manage products, customers, orders, and sales analysis.

This project demonstrates practical SQL concepts including relational database design, joins, aggregate functions, subqueries, customer spending analysis, revenue tracking, and sales reporting.
